# ForeignExchangeData

EUR/USD OHLCV data, 2-minute bars, for 2025 (January–December).

## Files

- `EURUSDT_2T_2025-MM.csv` — one file per month, 2-minute interval bars.

## Columns

| Column | Description |
|---|---|
| `datetime` | Bar timestamp (UTC) |
| `open` | Opening price |
| `high` | High price |
| `low` | Low price |
| `close` | Closing price |
| `volume` | Traded volume |
| `trade_count` | Number of trades |
